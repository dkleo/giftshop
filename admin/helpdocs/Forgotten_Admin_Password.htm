Forgotten Admin Password

If you have forgotten your admin password you will need to contact the website administrator.  If you are the administrator for your website, you will need to open your admin file located in the configuration folder and use the master username and password.

You can then login and reset the password for the account you want to login under.  Passwords cannot be emailed or displayed, because they are encrypted.  Make sure you set your password to something you can remember, but would be very difficult to guess.

Passwords that contain numbers and symbols, and that are at least 8 character long are recommended.
